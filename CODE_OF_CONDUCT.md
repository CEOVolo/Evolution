# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks
are not tolerated. We want a space where people learn in the open, disagree about ideas
without attacking each other, and build something strange and wonderful together.

## Scope

This applies in all project spaces — issues, pull requests, discussions — and when
representing the project in public spaces.

## Reporting

Report unacceptable behaviour to the maintainer (contact in the repository profile).
Reports are handled confidentially. The maintainer will review and respond, and may take
any action deemed appropriate, up to and including a ban from project spaces.

The full text of the Contributor Covenant 2.1 is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
